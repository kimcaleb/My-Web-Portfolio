Hycor Instrument Logbook

A full-stack application for keeping track of materials/external variables for any test runs performed on engineering and production prototype units developed at Hycor Biomedical. During the development and testing phase of IVD (In Vitro Diagnostics)/Medical Devices, it is important to log pretty much everything you do. This is because of two main reasons: passing internal/external audits for ISO cerfications/Meeting Company Compliance and GDP (Good Documentation Practices).Hycor has been using VB programmed Excel sheets to keep track of their instrument logs. Although it provided a basic layout for logging run details, there were numerous issues with this tool: The application did not have an external database, meaning if someone accidently deleted the excel sheet, all the data would be lost. Second, people often made numerous copies of one particular excel sheet, so it was hard to see which one was the original. Finally, the tool lagged a lot and often crashed when it reached the point where it contained hundreds of logs.
